# Tesla Showcase
Berisikan hasil tugas dengan fitur:

1. Menggunakan RecyclerView

2. Muncul halaman detail ketika salah satu item di tekan

3. Menggunakan style

## Intro
Merupakan viewer kendaraan Tesla, mulai dari model S, 3, X dan Roadster
